# Konceptu izpratnei nepieciešamo priekšnosacījumu relāciju noteikšana
- Šis repozitorijs satur visus rezultātus priekš maģistra darba ietvaros veiktajiem eksperimentiem, lai LVM iegūtie rezultāti būtu reproducējami.
- Direktorijs [scripts](https://github.com/HaraldsU/MD-prereq/tree/main/scripts) satur darbā pielietotos skriptus.
- Direktorijs [datasets](https://github.com/HaraldsU/MD-prereq/tree/main/datasets) satur darbā pielietotās datu kopas un LVM iegūtos rezultātus.
